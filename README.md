# KSP Crime Intelligence

Project scaffold for the KSP crime intelligence platform.

## Deployment note

The frontend no longer points at localhost. Set `VITE_API_BASE_URL` in your Netlify environment to the public backend URL, for example `https://your-backend.example.com`.

## Layout

- `backend/` FastAPI service stubs
- `frontend/` Vite React TypeScript app stubs
